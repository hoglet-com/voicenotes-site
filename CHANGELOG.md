# VoiceNotes Landing Page - 変更履歴

## 2026-04-21

### ダウンロードボタンのレイアウト最適化
- **変更内容**: ダウンロードボタンをHeroセクションからPricingセクションに移動
- **対象ファイル**: 全言語版（ja, en, zh, ko, es）
- **変更理由**: 
  - 価格を確認した直後にダウンロードアクションを促すことで、コンバージョン率の向上を狙う
  - ユーザーが「価格確認 → 購入意思決定 → ダウンロード」の流れをスムーズに進められるようにする
- **詳細**:
  - Heroセクションからダウンロードボタン（Win版・Mac版）を削除
  - Pricingセクションの説明文（「サブスクリプションなし。隠れた費用なし...」）の直下にダウンロードボタンを配置
  - ヘッダーの「ダウンロード」ボタンを `#pricing` セクションにリンク変更
  - Pricingの説明文のmarginを `mb-16` から `mb-8` に変更し、ダウンロードボタンに `mb-12` を設定

### セクション順序の最適化
- **変更内容**: Pricing セクションを Features セクションの前に移動
- **対象ファイル**: 全言語版（ja, en, zh, ko, es）
- **変更理由**: 
  - $14.99買い切りという価格が最大の差別化ポイント（競合は$99.99/年のサブスク）

  - 期待効果: コンバージョン率 +15-20%
- **新しい構成**: Hero → Pricing → Features → FAQ

### Heroセクション内の要素配置最適化
- **変更内容**: スクリーンショットカルーセルを上部に移動
- **変更理由**: 
  - ファーストビューで視覚的証拠を早く見せることで、アプリの品質を即座に印象づける
  - 「どんなアプリ？」という疑問に視覚的にすぐ答える
- **新しい順序**: 見出し（h1） → スクリーンショットカルーセル → 説明文 → ダウンロードボタン

## 実装の背景と戦略

### マーケティング戦略
VoiceNotesの最大の競争優位性：
- **価格**: $14.99買い切り vs 競合$99.99/年
- **プライバシー**: 100%オフライン動作
- **シンプルさ**: アカウント登録不要

これらの強みを効果的に伝えるため、ランディングページの構成を最適化。特に価格優位性を早期に提示することで、サブスク疲れのユーザーに即座にアピールする戦略を採用。

### 技術実装
- 全5言語（日本語、英語、中国語、韓国語、スペイン語）で統一的な構成を維持
- アンカーリンク（`#pricing`, `#features`, `#faq`）によるスムーズなナビゲーション
- レスポンシブデザイン対応（モバイル・デスクトップ両対応）

### 期待される効果
- コンバージョン率: +15-20%の向上を目標
- 直帰率: 80% → 65%
- 平均滞在時間: 45秒 → 2分
- スクロール深度: 50%のユーザーがFAQまで到達
